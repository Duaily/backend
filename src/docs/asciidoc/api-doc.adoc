== 1. 🎥 개요

본 문서는 Duaily REST API 문서입니다.
각 문서는 도메인을 기준으로 분리하였습니다.

문서에 대한 저작권은 ``구해줘 홈즈 ``팀에게 있으며, 무단으로 복제를 금지합니다.

=== 3-4. 본인이 작성한 빈 집 게시글 목록 조회 ( 페이징 )

- 월세인 경우

보증금과 월세 가격이 ( deposit, cost )로 제공됩니다.
